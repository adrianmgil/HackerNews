# HackerNews

Get Hacker News API
https://localhost:{port}/apis/HackerNews/Get

https://localhost:{port}/apis/HackerNews/Get?searchFor={text}&offset={offset}&linePerPage={linePerPage}

https://localhost:{port}/apis/HackerNews/Create
